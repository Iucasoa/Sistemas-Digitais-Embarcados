# 🖥️ Sistemas Digitais Embarcados

[![License](https://img.shields.io/badge/license-MIT-blue.svg)](LICENSE)
[![Made with](https://img.shields.io/badge/Made%20with-❤️-red.svg)](#)

Um repositório dedicado a materiais, códigos-fonte, exercícios e projetos desenvolvidos na disciplina de **Sistemas Embarcados**. Este acervo consolida conceitos fundamentais e práticas essenciais para o desenvolvimento de soluções embarcadas, contemplando desde o funcionamento do hardware até a implementação de software de baixo nível.

---

## 📋 Conteúdo do Repositório

### 🔧 [Códigos Arduino](./Códigos%20Arduino/)
Implementações práticas e exemplos de código utilizando a plataforma Arduino, abrangendo:
- Controle de periféricos e sensores
- Comunicação serial e interfaces
- Programação de microcontroladores
- Exemplos de projetos embarcados

### 🧠 [Projeto RTL - Perceptron](./Projeto%20RTL%20-%20Perceptron/)
Desenvolvimento de um **Perceptron em hardware** utilizando **Verilog/FPGA**, incluindo:
- Design de lógica digital (Register Transfer Level)
- Implementação de rede neural em FPGA
- Arquitetura de processamento de dados
- Simulação e validação de circuitos digitais

## 🚀 Como Usar Este Repositório

### Para os Códigos Arduino:
1. Abra o [Arduino IDE](https://www.arduino.cc/en/software)
2. Copie o código desejado da pasta `Códigos Arduino`
3. Carregue para sua placa Arduino
4. Consulte os comentários no código para entender a funcionalidade

### Para o Projeto Perceptron:
1. Tenha um ambiente de síntese de FPGA configurado (Vivado, Quartus, etc.)
2. Importe os arquivos `.v` da pasta `Projeto RTL - Perceptron`
3. Compile e sintetize o projeto
4. Faça o upload para sua placa FPGA

## 📚 Conceitos Abordados

### Sistemas Embarcados
- Microcontroladores e processadores
- Interrupções e timers
- Comunicação serial (UART, SPI, I2C)
- Programação de periféricos

### Sistemas Digitais
- Lógica combinacional e sequencial
- Máquinas de estados
- Circuitos síncronos
- Arquitetura de processadores

### Aplicações Práticas
- Redes neurais em hardware
- Processamento de sinais
- Automação e controle
- Prototipagem eletrônica

## 📖 Pré-requisitos

- Familiaridade com **Arduino IDE**
